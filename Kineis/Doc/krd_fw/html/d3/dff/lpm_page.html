‪LPM

This page is presenting the low-power-managment code specific to the Kineis reference design example based on STM32WL55xx uC (MGR_LPM).

It is in charge to configure the generic low power manager (Low Power Mode Manager (generic code)) to have its LPM (MgrLpm_LPM_t) working on STM32WL55xx uC.

It is also in charge to register the Kineis stack client so that the LPM manager knows which LPM to enter at any time of the Kineis protocol (idle phase, TX preparation, TX active, TX done).

lpm.c file contains overlaying functions to the mgr_lpm.c.

lpm_cli_kstk.c contains how to deduce the allowed LPM (MgrLpm_LPM_t) from Kineis stack ressources status (KNS_MAC_rsrcStatus_t).
