8 /**
9  * @page mgr_at_cmd_page AT commands manager
10  *
11  * The AT cmd manager is in charge of parsing an incoming data stream and extract AT commands. Then
12  * call the corresponding handler.
13  * This version of code is support AT command set v0.6.
14  *
15  * * mgr_at_cmd_list.c contains the **list of handlers for each AT cmd** (cf array
16  * \ref cas_atcmd_list_array)
17  * * mgr_at_cmd_common.h contains internal common utilities such as logging functions and the API
18  * to send AT responses over the console.
19  * * mgr_at_cmd.h is the main header file for AT cmd manager core.
20  * * mgr_at_cmd_list.h is the main entry point to get the list of suported commands
21  * (cf enum \ref atcmd_idx_t)
22  *
23  * @section mgr_at_cmd_subpages Sub-pages
24  *
25  * * @subpage user_data_page
