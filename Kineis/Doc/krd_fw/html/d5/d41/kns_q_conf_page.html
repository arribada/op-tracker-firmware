‪Kineis OS queue configuration

This page is presenting the configuration of OS queues for Kineis Oq abtstraction layer (KNS_Q). It should be fixed as per software architecture.

Typical parameters of a queue are:

  • ‪size of each queue (maximum number of items in queue)
  • ‪size of each element of the queue.

So far, the queues are listed in a simple enum (KNS_Q_handle_t) from lower-priority to higher-priority.
