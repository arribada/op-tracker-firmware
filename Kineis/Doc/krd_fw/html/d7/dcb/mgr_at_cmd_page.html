‪AT commands manager

The AT cmd manager is in charge of parsing an incoming data stream and extract AT commands. Then call the corresponding handler.

  • ‪mgr_at_cmd_list.c contains the list of handlers for each AT cmd (cf array cas_atcmd_list_array)
  • ‪mgr_at_cmd_common.h contains internal common utilities such as logging functions and the API to send AT responses over the console.
  • ‪mgr_at_cmd.h is the main header file for AT cmd manager core.
  • ‪mgr_at_cmd_list.h is the main entry point to get the list of suported commands (cf enum atcmd_idx_t)
